## 腳本簡介

- **script.tips.md**
  - 類型：Markdown 文件
  - 內容：提供 Windows PowerShell 常用指令示例
  - 功能：遞歸列舉檔案，包括絕對路徑、相對路徑和篩選特定副檔名（如 .ts）
- **extract-symbols.ts**
  - 類型：TypeScript 腳本
  - 依賴：使用 ts-morph 庫
  - 功能：從專案中提取符號資訊（如類、介面、函數），生成 Markdown 表格輸出
  - 參數：支援命令行參數指定輸出路徑
- **code-for-ai-analyze.ts**
  - 類型：TypeScript 模組
  - 主要類別：
    - FileScanner：掃描檔案
    - CodeAnalyzer：分析單一檔案結構和問題
    - ReportGenerator：生成項目報告
  - 用途：分析 TypeScript 檔案的品質指標，如行數、文檔覆蓋率和問題檢測
- **check-folder-size.cjs**
  - 類型：Node.js CommonJS 腳本
  - 功能：檢查指定根目錄下資料夾的子項目數量是否超過限制（預設 10）
  - 參數：支援 --root、--limit、--verbose
  - 輸出：報告超限的資料夾，適用於維護項目結構
