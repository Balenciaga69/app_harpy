# Difficulty-Scaler 模組

## 簡介

- 功能：負責根據遊戲進度、關卡、玩家狀態等參數，計算並提供難度係數（Difficulty Coefficient），作為敵人生成、關卡生成、獎勵分配等模組的依據。
- 特性：支援多種難度調整規則，並可根據需求擴充。
- 最後更新時間：2025-12-14。

## 輸入與輸出

### 主要輸入

- 當前章節、關卡層數、遊戲進度等遊戲狀態參數。
- 玩家等級、裝備強度、歷史表現等（可選）。

### 主要輸出

- 難度係數（DifficultyCoefficient）：一個數值或結構，供其他模組引用。

## 元件盤點

- DifficultyScaler：難度計算器，根據輸入參數計算難度係數。
- 介面層：定義 DifficultyScaler 相關的型別與契約，確保元件間一致性。

## 模組依賴誰?或被誰依賴?

Difficulty-Scaler 模組為基礎服務模組，不依賴其他遊戲邏輯模組。
被 Encounter（關卡生成）、EnemyGenerator（敵人生成）、Shop（商店）、PreCombat（戰鬥前準備）等模組依賴，用於調整生成參數與遊戲平衡。

## 開發者的碎碎念:

- 難度計算規則需可配置，方便日後調整平衡。
- 建議先實作最簡單的線性難度提升，後續再根據測試數據優化。
